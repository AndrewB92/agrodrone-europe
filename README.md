# AgroDrone Europe

A lightweight static website built with HTML, CSS and vanilla JavaScript.

## Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
* Web3Forms

## Structure

```text
/
├── index.html
├── style.css
├── script.js
├── assets/
├── pl/
│   └── index.html
├── robots.txt
└── sitemap.xml
```

## Development

No build tools or dependencies are required.

Run locally with any static server, for example:

```bash
python3 -m http.server
```

or simply open `index.html` in your browser.

## Deployment

The project can be deployed to any static hosting provider.

Tested with:

* GitHub Pages
* Cloudflare Pages
* Netlify
* Vercel (Static)

## Contact Forms

Both language versions use Web3Forms.

* German homepage (`/`)
* Polish homepage (`/pl`)

## Browser Support

Works in all modern browsers.
